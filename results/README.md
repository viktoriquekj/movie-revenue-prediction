# Results

This folder contains **model evaluation outputs**, **diagnostics**, **predictions**, and **feature-importance analyses** generated by the Movie Revenue Prediction pipeline.

**Note:**  
2025 predictions are forward-looking. Revenue for many 2025 movies is incomplete or not yet reported, and results for this year should be interpreted as **projections**, not ground-truth evaluation.

## Metrics

### **`metrics_summary.csv`**
Overall performance of the final model and baselines, including:

- RMSE / MAE in **log space**
- RMSE / MAE in **original revenue**
- R²  
- Metrics per year (2017–2025)

Useful for:
- Dashboard “Model Performance” page  
- Comparing to baseline  
- Trend analysis  

### **`rmse_by_year.csv`**
RMSE by **year** (train, validation, test).  

Use the following for evaluation:
- **2024 test** → True hold-out performance  
- **2025** → Forward-looking / not ground truth  

---

## Validation Predictions

### **`oof_predictions.csv`**
Out-of-fold predictions for all observations (train/val/test):

- `split`
- `year`
- `y_true_log`
- `y_pred_log`
- `baseline_mean_log`

Useful for:
- Residual diagnostics
- Drift detection
- Dashboard “Predicted vs Actual” page

---

## Final Deployment Output

### **`df_2025_predictions.csv`**
Predictions for **unreleased or ongoing 2025 movies** using the production Ensemble C model.

Columns include:
- `y_pred_log_revenue_C`
- `y_pred_revenue_C`
- All engineered features for traceability
